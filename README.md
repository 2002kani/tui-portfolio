# TUI Style Portfolio

A terminal-inspired personal portfolio built with React. Navigation works via arrow keys or the nav bar, its up to you.

![Homescreen](github/homescreen.png)


## Getting Started

```bash
npm install
npm run dev
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Arrow Left` | Previous page |
| `Arrow Right` | Next page |
